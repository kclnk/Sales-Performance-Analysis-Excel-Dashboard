# 📊 Sales Performance Dashboard – Excel

An advanced interactive Sales Performance Dashboard built entirely in **Microsoft Excel** using Pivot Tables, Power Query, formulas, and dynamic charts.

This project transforms raw sales data into a structured, decision-focused analytics dashboard.

---

## 🚀 Project Overview

This dashboard provides a complete analytical view of:

- Monthly & yearly revenue trends
- Customer distribution by country
- Product performance analysis
- Payment method insights
- Fulfillment rate tracking
- Sales team performance over time
- Average revenue per customer
- Average order value by country

The objective of this project is to demonstrate how Excel can be used as a powerful Business Intelligence tool — not just a spreadsheet.

---

## 🛠 Tools & Techniques Used

- Pivot Tables & Pivot Charts
- Power Query (Data Cleaning & Transformation)
- Dynamic Named Ranges
- Advanced Excel Formulas
- KPI Cards using custom formatting
- Slicers for interactive filtering
- Conditional Formatting
- Structured Tables

---

## 📊 Key KPIs

- Total Revenue
- Total Customers
- Products Sold
- Fulfilled vs Unfulfilled Orders
- Highest / Average / Lowest Monthly Revenue
- Average Revenue per Customer
- Average Order Value
- Orders by Payment Method

---

## 🎯 Dashboard Sections

### 1️⃣ Revenue Overview
Tracks total revenue, monthly trends, and yearly comparisons.

### 2️⃣ Customer Insights
Analyzes customer distribution by country and identifies top contributing regions.

### 3️⃣ Product Performance
Highlights top-performing products by cost and revenue contribution.

### 4️⃣ Payment Analytics
Breakdown of orders and revenue by payment method.

### 5️⃣ Fulfillment Analysis
Monitors fulfillment rate and unfulfilled orders.

### 6️⃣ Sales Team Performance
Compares sales representatives’ performance over time.
